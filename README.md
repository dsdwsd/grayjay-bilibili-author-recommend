# Grayjay Bilibili Plugin - 作者推荐版

基于官方 [futo-org/grayjay-plugin-bilibili](https://github.com/futo-org/grayjay-plugin-bilibili) 修改。

## 修改内容

视频推荐区只显示**当前视频作者**的其他视频，而非 B 站默认的混合推荐。

## 文件说明

- `BiliBiliScript.js` - 插件脚本（已编译）
- `BiliBiliConfig.json` - 插件配置
- `BiliBiliIcon.png` - 插件图标

## 安装

在 Grayjay 中通过自定义源 URL 安装本插件的 `BiliBiliConfig.json`。
